# FitSuperApp

FitSuperApp es una aplicación integral de seguimiento de actividad física para Android, creada con tecnologías modernas como Jetpack Compose, Kotlin y Room Database. Ayuda a los usuarios a gestionar sus rutinas de gimnasio y entrenamientos HIIT de manera eficiente.

## Características

- **Gestión de Rutinas de Gimnasio**: Seguimiento de ejercicios, series y progreso en tus rutinas de levantamiento de pesas.
- **Entrenamientos HIIT**: Selecciona y sigue rutinas de entrenamiento de intervalos de alta intensidad con temporizadores integrados.
- **Gestión de Datos**: Operaciones CRUD completas para ejercicios y rutinas.
- **Importación/Exportación CSV**: Realiza copias de seguridad o comparte tus datos de entrenamiento fácilmente mediante archivos CSV.
- **Persistencia Local**: Todos tus datos se guardan localmente en tu dispositivo usando la biblioteca Room Database.

## Stack Tecnológico

- **Lenguaje**: Kotlin
- **Framework de UI**: Jetpack Compose
- **Base de Datos**: Room Persistence Library
- **Arquitectura**: MVVM (Model-View-ViewModel)
- **Gestión de Dependencias**: Gradle (Kotlin DSL)

## Empezando

### Prerrequisitos

- Android Studio Koala o superior
- JDK 17+
- Android SDK 34+

### Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/FranciscoGarc/FitSuperApp.git
   ```
2. Abre el proyecto en Android Studio.
3. Sincroniza el proyecto con los archivos de Gradle.
4. Ejecuta la aplicación en un emulador o dispositivo físico.
